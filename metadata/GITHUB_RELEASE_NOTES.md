Draft GitHub release for the NODE replication materials.

This release is private/draft. It contains the minimal Python `node()` function
extracted from the supplementary information TeX source, citation metadata,
package manifest, and provenance notes.

Known gap: the inspected SI TeX contains the minimal allocation function, but
not a standalone runner that regenerates the paper figure, deterministic table,
and stochastic draw. Do not treat this as a final public replication release
until the intended license and reproduction scope are confirmed.
