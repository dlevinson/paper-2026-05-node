Draft GitHub release for the NODE replication materials.

This release is private/draft. It contains the minimal Python `node()` function
extracted from the supplementary information TeX source, a deterministic
toy-network example, citation metadata, package manifest, and provenance notes.

License: CC BY-SA 4.0, matching the published Findings page.

Scope note: the added example runs the deterministic toy-network allocation
from the paper text. The inspected SI TeX did not contain an end-to-end script
with plotting commands and stochastic-seed handling for every displayed paper
artifact, so this should be described as a minimal implementation plus example,
not as a full computational-reproduction archive.
